```python
def calc(option, *args):
    result = 0

    if option == 'add':
        for i in args:
            result += i
    
    elif option == 'mul':
        result = 1
        for i in args:
            result *= i

    elif option == 'sub':
        result = args[0]
        for i in args[1:]:
            result -= i

    elif option == 'div':       
        result = args[0]
        for i in args[1:]:
            result /= i
```

인턴하면서 했던 HMI처럼 인터페이스를 만들어주는 라이브러리가 있어서 그때 생각이 모락모락 나는 수업이였다.
